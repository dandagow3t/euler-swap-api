# Instalation
Copy `.env.template` to `.env` and set configuration

```
npm i
```

# Running

Dev server
```
npm run dev
```

Prod
```
npm run build
npm run start
```

# Lint
```
npm run lint       # check
npm run lint:fix   # fix
```
